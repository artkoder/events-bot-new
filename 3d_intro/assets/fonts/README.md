# Шрифты для 3D интро

Поместите следующие шрифты в эту директорию:

- **D-DIN** (d-din)
- **Akrobat**
- **Benzin**
- **Druk Cyr** (Druk Cur)

## Использование

Шрифты используются для высококачественного рендеринга текста на кубах в 3D сцене.
Текст не объёмный, но должен быть высокого разрешения.

## Формат

Шрифты должны быть в формате TTF или OTF для импорта в Blender.
