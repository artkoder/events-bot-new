# Туристическая метка (ручной режим)

Фича позволяет модераторам и администраторам вручную отмечать события как “интересно туристам / не интересно туристам”, выбирать причины и оставлять комментарий. Также доступен экспорт корпуса решений в JSONL.

## Документы

- User story: `docs/features/tourist-label/user-story.md`
- Исследование: `docs/features/tourist-label/research.md`
- Задачи/планы: `docs/features/tourist-label/tasks/README.md`

## Артефакты

- Экспорт `/tourist_export` (jsonl) складывать в `artifacts/parser-output/` или `artifacts/test-results/` (не коммитить).
