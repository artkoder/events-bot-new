# Source parsing: tasks

Здесь храним ссылки на задачи/планы/отчёты по фиче `source-parsing`.

## Типовые задачи

- Добавить новый источник (URL extraction → Kaggle/Playwright → нормализация → upsert → отчёт).
- Уточнить дедупликацию/матчинг существующих событий.
- Улучшить диагностику (диагностический режим, выгрузка логов, стабильность).
