# Классификация тем лекций

Для событий, у которых ещё нет сохранённых тем, вызывается LLM‑классификатор.

## Промпт

```
Определи 1–3 короткие темы для лекции. Ответь списком через запятую, без хэштегов, по-русски, в нижнем регистре.
Избегай дублирования и синонимов.
Текст: "{title}\n{description}"
```

## Нормализация

Результат сохраняется в событие и перед повторным использованием нормализуется по синонимам.

| Каноническая тема | Синонимы |
|-------------------|----------|
| искусство | культура |
| история россии | российская история |
| технологии | ит, тех |
| психология | mental health |
| урбанистика | город |
| литература | книги |
| кино | фильм, кинематограф |
| музыка | саунд, sound |
| бизнес | предпринимательство |

Словарь легко расширяется при необходимости.
